# Swalbard
A Sway + Pywal configuration






## Features

- Themed colors for your entire system based on chosen wallpapers
  - Terminal (kitty)
  - Waybar

- Customized shortcuts
  - Launch apps `Super + D`
  - Move focus: `Super + arrow keys`
  - Move focused window: `Super + shift + arrow keys`

# Installation


Before using dot files, make sure to at least have the necessary packages installed.

### Necessary packages

**Base packages:**

```
dnf install ...
```


**Terminal, browser and file manager:**

```
dnf install sway waybar kitty firefox  
```


# Setup

Copy /etc/sway/config to ~/.config/sway/config
